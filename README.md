Product Catalog API

This project is a RESTful backend api built with ASP.NET core and entity framework core.
It is built to represent a realistic backend service for managing a product catalog.


The API supports:

Creating, reading, updating, and deleting products (CRUD)
Pagination for scalable data access
Input validation using FluentValidation
Clean separation of concerns using Dependency Injection


The Stack used is:

C# / .NET 
Entity Framework Core
SQLite
FluentValidation
Dependency Injection



How to Run the Project

Restore dependencies:
dotnet restore

Apply database migrations:
dotnet ef database update

Run the application:
dotnet run

Test endpoints using:
curl

Invoke-RestMethod
Postman
